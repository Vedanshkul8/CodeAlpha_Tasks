# Exploratory Data Analysis (EDA) - Superstore Dataset
Project 2:
## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Sample Superstore dataset to uncover business insights, identify trends, detect anomalies, and validate assumptions using Python and data visualization techniques.

The analysis focuses on understanding sales performance, profitability, customer segments, regional performance, discount impact, and potential data quality issues.

## Objectives

* Explore dataset structure and data types.
* Identify missing values and duplicate records.
* Analyze sales and profit across categories, regions, and customer segments.
* Detect trends, patterns, and anomalies.
* Test business hypotheses using statistical analysis and visualizations.
* Identify potential data issues for further investigation.

## Dataset Information

* Dataset: Sample Superstore
* Records: 9,994
* Features: 21
* Domain: Retail Sales

### Key Variables

* Sales
* Profit
* Discount
* Quantity
* Category
* Sub-Category
* Segment
* Region
* State
* Order Date

### Correlation Analysis

* Relationship among Sales, Profit, Quantity, and Discount

## Key Findings

* Technology category generated the highest sales.
* Consumer segment contributed the largest share of revenue.
* Higher discounts negatively impacted profitability.
* Certain sub-categories resulted in losses.
* Significant outliers were identified in Sales and Profit.
* Regional performance varied considerably across the business.
* Sales exhibited fluctuations over time.

## Recommendations

* Reduce excessive discounts on low-margin products.
* Focus on profitable categories and regions.
* Investigate loss-making sub-categories.
* Monitor unusual transactions and outliers.
* Use insights to improve inventory and pricing decisions.

## Conclusion

The EDA successfully identified important business insights, patterns, and potential issues within the Superstore dataset. These findings can support better decision-making and serve as a foundation for advanced analytics and predictive modeling.
