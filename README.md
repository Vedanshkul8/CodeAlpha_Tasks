# Superstore Sales & Profit Analysis Dashboard

## Overview
This project was created as part of the CodeAlpha Data Analytics Internship (Task 3: Data Visualization). Using Tableau, I developed an interactive dashboard to analyze sales, profit, regional performance, and customer segments from the Superstore dataset.

## Tools Used
- Tableau
- Superstore Dataset (CSV)

## Dashboard Features
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Sales vs Profit Analysis
- Sales by State Map
- Interactive Filters

## Outcome
The dashboard helps identify business trends, profitable regions, and category performance through clear and interactive visualizations.
